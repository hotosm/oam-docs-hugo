    Tiler

    The Tiler is the tool and API to generate tiled map service endpoints from OAM imagery.

    See oam-server-tiler, oam-server-api, and oam-server-cli for details on the OAM Tiler components.

    The OpenAerialMap Mosaic has documentation for filtering the mosaic as Swagger.
